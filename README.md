# 🚀 Sakthivel Madhu – Portfolio

This is my personal developer portfolio showcasing my experience in building **enterprise-grade backend systems**, including Purchase, Vendor, and Finance platforms.

The portfolio is built using **HTML, CSS, and JavaScript** with a modern **Gradient UI**, animated sliders, and a responsive layout.

---

## 🌟 Features

- Gradient Modern UI
- Glassmorphism design
- Fully responsive (mobile + desktop)
- Skills slider (auto-scroll + arrows)
- Projects slider (highlighted enterprise systems)
- Profile hero section
- Smooth animations
- Contact form integration
